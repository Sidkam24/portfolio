## 【概要】
　・福岡教育大学卒  
　・中学校技術科教師としてプログラミング教育を行った。  
　　そこではHTML, CSS, JavaScriptをメインに授業を行った。  
　・ICT主任として率先して先生にも子どもにもICT活用を推進した。  
　・サッカー部顧問として５年間指導した。  
　・制作物すべて、実際に授業で活用した。
